# @webstir-io/module-contract

TypeScript definitions and JSON Schemas describing the Webstir module-provider contract. Providers implement this API to integrate build pipelines with the Webstir CLI.

## Install

```bash
npm install @webstir-io/module-contract
```

## Development

```bash
npm ci
npm run build
npm test
```

## Scripts

| Command        | Description                               |
| -------------- | ----------------------------------------- |
| `npm run build` | Compile TypeScript and emit schemas.       |
| `npm test`      | Build then exercise contract validation.   |
| `npm run clean` | Remove build artifacts.                    |
